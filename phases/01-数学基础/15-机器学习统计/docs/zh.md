# 机器学习统计学

> 统计学能告诉你模型是真的有效，还是仅仅靠运气。

**类型：** 构建
**语言：** Python
**前置要求：** Phase 1，课程 06（概率与分布）、07（贝叶斯定理）
**时间：** 约 120 分钟

## 学习目标

- 从零实现描述性统计、皮尔逊/斯皮尔曼相关系数和协方差矩阵
- 执行假设检验（t 检验、卡方检验）并正确解释 p 值和置信区间
- 使用自助法（Bootstrap）重采样，在不假设分布的情况下为任意指标构造置信区间
- 使用效应量指标区分统计显著性与人实际意义

## 问题所在

你训练了两个模型。模型 A 在测试集上得分 0.87。模型 B 得分 0.89。你部署了模型 B。三周后，生产指标反而变差了。发生了什么？

模型 B 实际上并没有真正优于模型 A。0.02 的差异是噪声。你的测试集太小，或者方差太大，或者两者兼有。你发布的是披着改进外衣的随机性。

这种情况无处不在。Kaggle 排行榜震荡。论文无法复现。A/B 测试仅凭几百个样本就宣布获胜者。根本原因总是一样的：有人跳过了统计学。

统计学给了你区分信号与噪声的工具。它告诉你差异是否真实，你应该有多大的把握，以及在信任一个结果之前需要多少数据。每个 ML 流水线、每个模型比较、每个实验都需要统计学。没有它，你只是在猜测。

## 核心概念

### 描述性统计：总结你的数据

在任何建模之前，你需要知道你的数据长什么样。描述性统计将数据集压缩成几个能捕捉数据形态的数字。

**集中趋势度量**回答"中间在哪里？"

```
均值：   所有值的和 / 数量
         mu = (1/n) * sum(x_i)

中位数： 排序后的中间值
         对异常值稳健。如果你有 [1, 2, 3, 4, 1000]，均值是 202
         但中位数是 3。

众数：   出现频率最高的值
         对分类数据有用。对连续数据，很少有信息量。
```

均值是平衡点。中位数是半程标记。当它们偏离时，你的分布是偏态的。收入分布的均值远大于中位数（亿万富翁造成右偏）。训练时的损失分布通常均值远小于中位数（简单样本造成左偏）。

**离散程度度量**回答"数据有多分散？"

```
方差：     偏离均值的平均平方偏差
           sigma^2 = (1/n) * sum((x_i - mu)^2)

标准差：   方差的平方根
           sigma = sqrt(sigma^2)
           与数据单位相同，因此更易解释。

极差：     最大值 - 最小值
           对异常值敏感。仅单独使用几乎毫无意义。

四分位距： Q3 - Q1（第75百分位 - 第25百分位）
           中间 50% 数据的范围。
           对异常值稳健。用于箱线图和异常值检测。
```

**百分位数**将排序后的数据分成 100 等份。第 25 百分位（Q1）意味着 25% 的值落在这个点以下。第 50 百分位就是中位数。第 75 百分位是 Q3。

```
延迟监控示例：
  P50 = 中位数延迟        （典型用户体验）
  P95 = 第95百分位       （差但不是最坏情况）
  P99 = 第99百分位       （尾部延迟，通常是中位数的10倍）
```

在 ML 中，你关心百分位数用于：推理延迟、预测置信度分布、理解误差分布。一个平均误差低但 P99 误差极差的模型可能在安全关键应用中毫无用处。

**样本统计量 vs 总体统计量。** 从样本计算方差时，除以 (n-1) 而不是 n。这就是贝塞尔校正（Bessel's correction）。它补偿了样本均值不是真实总体均值的 fact。用 n 作除数会系统性地低估真实方差。用 (n-1)，估计是无偏的。

```
总体方差： sigma^2 = (1/N) * sum((x_i - mu)^2)
样本方差：  s^2     = (1/(n-1)) * sum((x_i - x_bar)^2)
```

实践中：如果 n 很大（数千个样本），差异可以忽略。如果 n 很小（几十个样本），差异很重要。

### 相关性：变量如何一起移动

相关性衡量两个变量之间线性关系的强度和方向。

**皮尔逊相关系数**衡量线性相关性：

```
r = sum((x_i - x_bar)(y_i - y_bar)) / (n * s_x * s_y)

r = +1： 完美正线性关系
r = -1： 完美负线性关系
r =  0： 无线性关系（但可能存在非线性关系！）

范围：[-1, 1]
```

皮尔逊假设关系是线性的且两个变量大致服从正态分布。它对异常值敏感。一个极端点可以把 r 从 0.1 拉到 0.9。

**斯皮尔曼秩相关**衡量单调相关性：

```
1. 将每个值替换为其秩（1, 2, 3, ...）
2. 在这些秩上计算皮尔逊相关

斯皮尔曼能捕捉任何单调关系，而不仅仅是线性的。
如果 y = x^3，皮尔逊给出 r < 1 但斯皮尔曼给出 rho = 1。
```

**何时使用各自：**

```
皮尔逊：    两个变量都是连续且大致正态的。
            你特别关心线性关系。
            没有极端异常值。

斯皮尔曼：  有序数据（排名、评分）。
            数据不是正态分布的。
            你怀疑是单调但非线性关系。
            存在异常值。
```

**黄金法则：** 相关性不意味着因果关系。冰淇淋销量和溺水死亡相关，因为两者都在夏季增加。你的模型准确率和参数数量相关，但增加参数并不会自动提高准确率（参见：过拟合）。

### 协方差矩阵

两个变量之间的协方差衡量它们如何一起变化：

```
Cov(X, Y) = (1/n) * sum((x_i - x_bar)(y_i - y_bar))

Cov(X, Y) > 0：  X 和 Y 倾向于一起增加
Cov(X, Y) < 0：  当 X 增加时，Y 倾向于减少
Cov(X, Y) = 0： 无线性共同移动
```

对于 d 个特征，协方差矩阵 C 是一个 d × d 矩阵，其中 C[i][j] = Cov(特征_i, 特征_j)。对角线元素 C[i][i] 是每个特征的方差。

```
C = | Var(x1)      Cov(x1,x2)  Cov(x1,x3) |
    | Cov(x2,x1)  Var(x2)      Cov(x2,x3) |
    | Cov(x3,x1)  Cov(x3,x2)  Var(x3)     |

性质：
  - 对称：C[i][j] = C[j][i]
  - 半正定：所有特征值 >= 0
  - 对角线 = 方差
  - 非对角线 = 协方差
```

**与 PCA 的联系。** PCA 对协方差矩阵进行特征分解。特征向量是主成分（最大方差方向）。特征值告诉你每个主成分捕获了多少方差。这正是课程 10 所讲的，但现在你明白了为什么协方差矩阵是正确分解的对象：它编码了数据中所有成对的线性关系。

**与相关的联系。** 相关矩阵是标准化变量的协方差矩阵（每个变量除以其标准差）。相关性标准化了协方差，使所有值落在 [-1, 1] 范围内。

### 假设检验

假设检验是在不确定性下做决策的框架。你从某个主张开始，收集数据，然后确定数据是否与该主张一致。

**设置：**

```
零假设 (H0)：  默认假设，通常是"无效果"
备择假设 (H1)：你想证明的内容

示例：
  H0：模型 A 和模型 B 准确率相同
  H1：模型 B 的准确率高于模型 A
```

**p 值**是在 H0 为真的假设下，看到与你观察到的一样极端或更极端的数据的概率。它不是 H0 为真的概率。这是统计学中最常见的误解。

```
p-value = P(数据如此极端 | H0 为真)

如果 p-value < alpha（通常是 0.05）：
    拒绝 H0。结果是"统计显著的"。
如果 p-value >= alpha：
    无法拒绝 H0。你没有足够的证据。
    这并不意味 H0 为真。
```

**置信区间**给出参数的合理值范围：

```
均值的 95% 置信区间：
    x_bar +/- z * (s / sqrt(n))

其中 z = 1.96 表示 95% 置信度

解释：如果你重复这个实验很多次，95% 的计算区间会包含真实均值。
这并不意味着真实均值有 95% 的概率在这个特定区间内。
```

置信区间的宽度告诉你精度。宽区间意味着高度不确定性。窄区间意味着你的估计是精确的（但如果你的数据有偏倚，不一定准确）。

### t 检验

t 检验比较均值。有几种类型。

**单样本 t 检验：** 总体均值是否与假设值不同？

```
t = (x_bar - mu_0) / (s / sqrt(n))

自由度 = n - 1
```

**双样本 t 检验（独立）：** 两组均值是否不同？

```
t = (x_bar_1 - x_bar_2) / sqrt(s1^2/n1 + s2^2/n2)

这是 Welch's t 检验，不假设方差相等。
除非你有特定理由要求方差相等，否则总是使用 Welch's。
```

**配对 t 检验：** 当测量值成对出现时（同一模型在相同数据划分上评估）：

```
计算每对的 d_i = x_i - y_i
然后对 d_i 值执行单样本 t 检验，针对 mu_0 = 0
```

在 ML 中，配对 t 检验很常见：你对两个模型使用相同的 10 折交叉验证并成对比较它们的分数。

### 卡方检验

卡方检验检查观察到的频率是否与期望频率匹配。对分类数据有用。

```
chi^2 = sum((observed - expected)^2 / expected)

示例：语言模型的输出分布在各类别上是否匹配训练分布？

类别      观察值   期望值
正面        120      100
负面         80      100
chi^2 = (120-100)^2/100 + (80-100)^2/100 = 4 + 4 = 8

自由度为 1 时，chi^2 = 8 给出 p < 0.005。
差异是显著的。
```

### ML 模型的 A/B 测试

ML 中的 A/B 测试不同于网页 A/B 测试。模型比较有其特定挑战：

```
1. 相同测试集：    两个模型必须在相同数据上评估。
                   不同的测试集使比较毫无意义。

2. 多个指标：       仅准确率不够。你需要精确率、
                   召回率、F1、延迟和公平性指标。

3. 方差：          使用交叉验证或自助法来估计
                   每个指标的方差，而不仅仅是点估计。

4. 数据泄漏：      如果测试集在模型选择期间被使用，
                   你的比较是有偏的。留出最终测试集。
```

**程序：**

```
1. 定义指标和显著性水平（alpha = 0.05）
2. 在相同的 k 折交叉验证划分上运行两个模型
3. 收集成对分数：[(a1, b1), (a2, b2), ..., (ak, bk)]
4. 计算差异：d_i = b_i - a_i
5. 对差异执行配对 t 检验
6. 检查：均值差异是否显著不同于 0？
7. 计算均值差异的置信区间
8. 计算效应量（Cohen's d）以判断实际意义
```

### 统计显著性 vs 实际显著性

结果可能统计显著但实际毫无意义。数据足够多时，即使一个微不足道的差异也会变得统计显著。

```
示例：
  模型 A 准确率：0.9234
  模型 B 准确率：0.9237
  n = 1,000,000 测试样本
  p-value = 0.001

统计显著？ 是。
实际显著？ 0.03% 的改进不值得部署新模型的工程成本。
```

**效应量**量化差异有多大，与样本量无关：

```
Cohen's d = (mean_1 - mean_2) / pooled_std

d = 0.2：  小效应
d = 0.5：  中效应
d = 0.8：  大效应
```

总是同时报告 p 值和效应量。p 值告诉你差异是否真实。效应量告诉你差异是否重要。

### 多重比较问题

当你检验多个假设时，有些会偶然"显著"。如果你在 alpha = 0.05 下检验 20 个事物，即使没有任何真实差异，你也会期望 1 个假阳性。

```
P(至少一个假阳性) = 1 - (1 - alpha)^m

m = 20 次检验，alpha = 0.05：
P(假阳性) = 1 - 0.95^20 = 0.64

你有 64% 的概率至少有一个假阳性。
```

**Bonferroni 校正：** 用检验数量除以 alpha。

```
调整后 alpha = alpha / m = 0.05 / 20 = 0.0025

仅当 p-value < 0.0025 时拒绝 H0。
保守但简单。当检验独立时有效。
```

在 ML 中，这在比较跨多个指标的模型、测试许多超参数配置或在多个数据集上评估时很重要。

### 自助法（Bootstrap）

自助法通过对数据进行有放回重采样来估计统计量的抽样分布。不需要对底层分布做任何假设。

**算法：**

```
1. 你有 n 个数据点
2. 有放回地抽取 n 个样本（有些点出现多次，有些完全没有）
3. 在这个自助样本上计算你的统计量
4. 重复 B 次（通常 B = 1000 到 10000）
5. 自助统计量的分布近似于抽样分布
```

**自助置信区间（百分位数法）：**

```
排序 B 个自助统计量
95% CI = [第2.5百分位, 第97.5百分位]
```

**为什么自助法对 ML 重要：**

```
- 测试集准确率是一个点估计。自助法给你置信区间。
- 你不能假设指标分布是正态的（尤其是 AUC、F1、Precision@K）。
- 自助法适用于任何统计量：中位数、两个均值之比、
  两个模型 AUC 之差。
- 不需要封闭形式的公式。
```

**用于模型比较的自助法：**

```
1. 你有模型 A 和模型 B 在相同测试集上的预测
2. 对于每次自助迭代：
   a. 有放回地重采样测试索引
   b. 在重采样的集合上计算 metric_A 和 metric_B
   c. 存储 diff = metric_B - metric_A
3. 差异的 95% CI：
   [diff 的第2.5百分位, diff 的第97.5百分位]
4. 如果 CI 不包含 0，差异是显著的
```

这比配对 t 检验更稳健，因为它不做任何分布假设。

### 参数检验 vs 非参数检验

**参数检验**假设数据服从特定分布（通常是正态分布）：

```
t 检验：        假设数据正态分布（或大 n 时 CLT 成立）
ANOVA：          假设正态性和等方差
Pearson r：      假设二元正态性
```

**非参数检验**不做任何分布假设：

```
Mann-Whitney U：    比较两组（替代独立 t 检验）
Wilcoxon 符号秩：   比较配对数据（替代配对 t 检验）
Spearman rho：      在秩上计算相关（替代皮尔逊）
Kruskal-Wallis：   比较多组（替代 ANOVA）
```

**何时使用非参数：**

```
- 样本量小（n < 30）且数据明显非正态
- 有序数据（评分、排名）
- 有无法去除的重异常值
- 分布偏斜
```

**何时使用参数：**

```
- 样本量大（CLT 使检验统计量近似正态）
- 数据大致对称且无极端异常值
- 统计功效更强（更好检测真实差异）
```

在 ML 实验中，你通常样本量小（5 或 10 个交叉验证折），因此非参数检验（如 Wilcoxon 符号秩）通常比 t 检验更合适。

### 中心极限定理：实际意义

中心极限定理（CLT）表明，随着 n 增大，样本均值的分布趋近正态分布，无论底层总体分布是什么。

```
如果 X_1, X_2, ..., X_n 是独立同分布的，均值为 mu，方差为 sigma^2：

    X_bar ~ Normal(mu, sigma^2 / n)    随着 n -> 无穷

大多数情况下 n >= 30 即可。
对于高度偏斜的分布，可能需要 n >= 100。
```

**为什么这在 ML 中很重要：**

```
1. 为聚合指标上的置信区间和 t 检验提供依据
2. 解释了跨交叉验证折平均化给出稳定估计，
   即使单个折差异很大
3. 小批量梯度下降有效，因为批量上的平均梯度
   近似真实梯度（CLT 在起作用）
4. 集成方法：从许多模型平均预测比任何单一模型
   给出更稳定的输出
```

**CLT 不能做到的事情：**

```
- 不能使你的数据变成正态。它使样本均值的分布正态。
- 对无限方差的重尾分布无效（柯西分布）。
- 不适用于非独立数据（未经校正的时间序列）。
```

### ML 论文中常见的统计错误

1. **在训练集上测试。** 保证过拟合。始终留出模型在训练期间从未见过的数据。

2. **没有置信区间。** 仅报告单一准确率数字而不带不确定性，使结果不可复现和验证。

3. **忽略多重比较。** 测试 50 个配置并报告最好的一个而不做校正，会膨胀假阳性率。

4. **混淆统计显著性和实际显著性。** 在 0.01% 准确率提升上 p 值为 0.001 没有意义。

5. **在不平衡数据上使用准确率。** 在 99% 为负类的数据集上准确率 99% 意味着模型什么都没学到。使用精确率、召回率、F1 或 AUC。

6. **选择性报告指标。** 只报告模型获胜的指标。诚实评估应报告所有相关指标。

7. **训练/测试划分间泄漏信息。** 在划分前归一化，或用未来数据预测过去。

8. **测试集小且无方差估计。** 在 100 个样本上评估并声称 2% 的改进是噪声，不是信号。

9. **假设数据独立而实际不独立。** 来自同一患者的医学图像，来自同一文档的多个句子。一组内的观察是相关的。

10. **P-hacking。** 尝试不同的检验、子集或排除标准，直到得到 p < 0.05。结果是搜索的人为产物。

## 关键术语

| 术语 | 定义 |
|---|---|
| 均值 | 值的和除以数量。对异常值敏感。 |
| 中位数 | 排序数据中的中间值。对异常值稳健。 |
| 标准差 | 方差的平方根。以原始单位度量离散程度。 |
| 百分位数 | 给定百分比的数据落在其下的值。 |
| 四分位距 | Q3 减 Q1。中间 50% 数据的范围。 |
| 皮尔逊相关 | 衡量两个变量之间的线性相关性。范围 [-1, 1]。 |
| 斯皮尔曼相关 | 使用秩来衡量单调相关性。 |
| 协方差矩阵 | 所有特征之间成对协方差的矩阵。 |
| 零假设 | 默认假设：无效果或无差异。 |
| p 值 | 在零假设为真时观察到如此极端的数据的概率。 |
| 置信区间 | 参数在给定置信水平下的合理值范围。 |
| t 检验 | 使用 t 分布检验均值是否有显著差异。 |
| 卡方检验 | 检验观察到的频率是否与期望频率不同。 |
| 效应量 | 与样本量无关的差异幅度。Cohen's d 是常见指标。 |
| Bonferroni 校正 | 除以检验数量以控制假阳性的显著性阈值。 |
| 自助法 | 有放回重采样以估计抽样分布。 |
| I 类错误 | 假阳性。拒绝为真的 H0。 |
| II 类错误 | 假阴性。未拒绝为假的 H0。 |
| 统计功效 | 正确拒绝假的 H0 的概率。功效 = 1 - II 类错误率。 |
| 中心极限定理 | 样本均值随着样本量增长收敛到正态分布。 |
| 参数检验 | 假设数据的特定分布（通常是正态分布）。 |
| 非参数检验 | 不做分布假设。适用于秩或符号。 |
