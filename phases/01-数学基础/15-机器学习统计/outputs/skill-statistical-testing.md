# ML 统计检验

在比较模型、运行 A/B 实验或验证结果时如何选择正确的检验方法。

## 决策检查清单

1. 你在比较什么？均值、比例、分布，还是相关性？
2. 有多少组？单样本与参考值、双组，还是多组？
3. 观察值是配对（相同测试集、相同折）还是独立的？
4. 数据是正态分布的吗？如果 n < 30 且不明显正态，使用非参数方法。
5. 数据是连续、有序还是分类的？
6. 你要运行多少次检验？如果超过一次，应用校正。

## 决策树

```text
比较均值？
  双组？
    配对（相同数据划分）？--> 配对 t 检验（非正态用 Wilcoxon 符号秩）
    独立？ --> Welch's t 检验（非正态用 Mann-Whitney U）
  多组？
    配对？ --> 重复测量 ANOVA（非正态用 Friedman 检验）
    独立？ --> 单因素 ANOVA（非正态用 Kruskal-Wallis）

比较比例？
  双组？ --> 卡方检验或 Fisher 精确检验（样本小时）
  多组？ --> 卡方检验

比较分布？
  一个分布是参考分布？ --> Kolmogorov-Smirnov 检验
  两个都是经验分布？ --> 双样本 KS 检验

测量相关性？
  两个都连续且大致正态？ --> 皮尔逊相关
  有序或非正态？ --> 斯皮尔曼秩相关
  分类 × 分类？ --> 卡方独立性检验

运行多次检验？
  应用 Bonferroni 校正：adjusted_alpha = alpha / 检验数量
  或使用 Holm-Bonferroni（较不保守，仍控制族错误率）
```

## 何时使用每种检验

| 检验 | 数据类型 | 假设 | ML 使用场景 |
|---|---|---|---|
| 配对 t 检验 | 连续，配对 | 差值正态 | 比较同一 k 折上的两个模型 |
| Wilcoxon 符号秩 | 连续/有序，配对 | 无（非参数） | 比较两个模型，小 k（5-10 折） |
| Welch's t 检验 | 连续，独立 | 大致正态 | 比较两个独立数据集上的模型 |
| Mann-Whitney U | 连续/有序，独立 | 无 | 比较延迟分布 |
| ANOVA | 连续，3+ 组 | 正态，等方差 | 比较多个模型架构 |
| Kruskal-Wallis | 连续/有序，3+ 组 | 无 | 比较多个模型，非正态指标 |
| 卡方检验 | 分类计数 | 期望频数 >= 5 | 比较类别分布、混淆矩阵 |
| Fisher 精确检验 | 分类计数 | 小样本 | 稀有事件比较 |
| KS 检验 | 连续 | 无 | 检查预测是否符合期望分布 |
| Bootstrap CI | 任意统计量 | 无 | AUC、F1、任意指标的置信区间 |
| McNemar 检验 | 配对二分类 | 无 | 比较同一测试集上的两个分类器 |

## 模型比较流程

1. 实验前定义指标和显著性水平（alpha = 0.05）。
2. 在相同的 k 折交叉验证划分上运行两个模型（k = 5 或 10）。
3. 收集成对分数：(a_1, b_1), (a_2, b_2), ..., (a_k, b_k)。
4. 计算差异：d_i = b_i - a_i。
5. 运行配对检验（k <= 10 用 Wilcoxon，k > 10 或差值正态用配对 t 检验）。
6. 报告：p 值、均值差异、95% 置信区间、效应量（Cohen's d）。
7. 如果 p < alpha 且效应量有实际意义，差异是真实且值得行动的。

## 常见错误

- 当数据配对时使用独立检验。如果两个模型在相同测试折上评估，必须使用配对检验。独立检验丢弃了配对信息，失去统计功效。
- 报告 p < 0.05 而不报告效应量。统计显著但仅 0.1% 准确率提升不值得部署。总要计算 Cohen's d 或原始均值差异。
- 在不同测试集上比较模型。测试集必须对两个模型完全相同。不同的测试集使比较毫无意义。
- 运行 20 次比较而不做 Bonferroni 校正就报告最好的一个。在 alpha = 0.05 下 20 次检验，期望有 1 次偶然"显著"。
- 在不平衡数据上使用准确率。在 99% 多数类上，平凡分类器达到 99%。使用 F1、精确率-召回率 AUC 或 Matthews 相关系数。
- 将交叉验证折作为独立样本。它们共享训练数据，违反独立性假设。校正的重采样 t 检验考虑了这一点。

## 快速参考：效应量解释

| Cohen's d | 解释 |
|---|---|
| 0.2 | 小效应 |
| 0.5 | 中效应 |
| 0.8 | 大效应 |
| > 1.0 | 非常大效应 |

| 应报告内容 | 为什么 |
|---|---|
| p 值 | 差异是否真实？ |
| 置信区间 | 差异可能有多大？ |
| 效应量（Cohen's d） | 差异是否有意义？ |
| 样本量（n 或 k 折） | 我们能信任这个结果吗？ |
