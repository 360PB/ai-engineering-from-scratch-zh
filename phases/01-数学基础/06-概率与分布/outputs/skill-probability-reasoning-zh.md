# 概率分布选择

如何在建模数据、设计损失函数或设置先验时挑选合适的分布。

## 决策清单

1. 结果是离散（类别、计数）还是连续（测量值、分数）？
2. 结果是否有界（如 [0, 1]）还是无界？
3. 有多少种可能结果？两种？k 种？无限种？
4. 数据对称还是偏斜？
5. 事件独立还是相关？
6. 你在建模的是比率、计数、比例还是测量值？

## 分布决策树

```
变量是离散的吗？
  是 --> 只有 2 种结果？ --> 伯努利 (p)
     |    k 种结果，一次试验？ --> 分类 (p1...pk)
     |    k 种结果，n 次试验？ --> 多项分布 (n, p1...pk)
     |    n 次试验中的成功计数？ --> 二项分布 (n, p)
     |    每区间的事件计数？ --> 泊松 (lambda)
     |    直到首次成功的试验次数？ --> 几何分布 (p)
     |    直到 r 次成功的试验次数？ --> 负二项分布 (r, p)
  否 --> 对称、钟形？ --> 正态 (mu, sigma)
     |   正值、右偏？ --> 对数正态或指数分布
     |   有界于 [0, 1]？ --> Beta (alpha, beta)
     |   正值、形状灵活？ --> Gamma (alpha, beta)
     |   事件之间的时间？ --> 指数分布 (lambda)
     |   需要重尾？ --> 学生 t (nu) 或柯西分布
     |   多元、钟形？ --> 多元正态
     |   在单纯形上（和为 1）？ --> 狄利克雷 (alpha)
```

## 将现实 ML 场景映射到分布

| 场景 | 分布 | 参数 |
|---|---|---|
| 二元分类输出 | 伯努利 | p = sigmoid(logit) |
| 多类分类输出 | 分类 | p = softmax(logits) |
| 语言模型中的 token 预测 | 词表上的分类 | p 来自 softmax |
| 像素强度（归一化） | Beta 或均匀 [0, 1] | 取决于图像统计 |
| 文档中的词数 | 泊松 | lambda = 平均词数 |
| 用户请求间隔时间 | 指数分布 | lambda = 请求率 |
| 测量误差 | 正态 | mu = 0, sigma 从数据估计 |
| 权重初始化 | 正态或均匀 | Kaiming/Xavier 规则 |
| VAE 隐空间先验 | 标准正态 | mu = 0, sigma = 1 |
| 比例上的贝叶斯先验 | Beta | alpha, beta 来自先验信念 |
| 类别权重上的贝叶斯先验 | 狄利克雷 | alpha 向量 |
| 回归目标中的噪声 | 正态 | mu = 0, sigma 估计 |
| 抗异常值回归 | 学生 t | 低自由度 |
| 持续时间/寿命建模 | 韦伯或 Gamma | 形状和尺度 |
| 每篇文档的主题分布 (LDA) | 狄利克雷 | alpha < 1 用于稀疏 |

## 分布用错的情况

- 数据有硬下界（如价格、距离）时使用正态。正态给负值分配非零概率。改用对数正态或 gamma。
- 方差不等于均值时使用泊松。泊松假设均值 = 方差。若方差 > 均值，用负二项分布。
- 多类问题使用伯努利。伯努利严格是二元的。k > 2 时用分类分布。
- 观测相关时假设独立。时间序列、空间数据和分组数据违反独立性。用自回归或分层模型。

## 常见错误

- 把 PDF 值当作概率。PDF 可以超过 1。概率来自在区间上积分 PDF。
- 忘记 softmax 输出是分类概率，不是独立的伯努利概率。它们按构造和为 1。
- 有领域知识时使用均匀先验。如果先验选择得当，信息性先验能在不引入偏差的情况下减少方差。
- 把对数概率当作概率。对数概率总是负的（或零）。它们之和不为 1。

## 速查：分布性质

| 分布 | 支撑集 | 均值 | 方差 | 关键性质 |
|---|---|---|---|---|
| 伯努利(p) | {0, 1} | p | p(1-p) | 最简单的离散分布 |
| 二项分布(n, p) | {0..n} | np | np(1-p) | n 个伯努利之和 |
| 泊松(lam) | {0, 1, 2, ...} | lam | lam | 均值 = 方差 |
| 正态(mu, s^2) | (-inf, inf) | mu | s^2 | 给定均值/方差下最大熵 |
| 指数分布(lam) | [0, inf) | 1/lam | 1/lam^2 | 无记忆性 |
| Beta(a, b) | [0, 1] | a/(a+b) | ab/((a+b)^2(a+b+1)) | 二项分布的共轭先验 |
| Gamma(a, b) | (0, inf) | a/b | a/b^2 | 泊松的共轭先验 |
| 狄利克雷(alpha) | 单纯形 | alpha_i/sum | (见公式) | 分类分布的共轭先验 |
