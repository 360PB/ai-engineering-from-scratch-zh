# 采样策略选择

如何为文本生成、贝叶斯推理、蒙特卡洛估计和训练选择正确的采样方法。

## 决策检查清单

1. 你是在生成输出（文本、图像）还是估计数量（积分、期望）？
2. 你能直接从目标分布采样，还是只能评估其密度？
3. 目标分布是离散还是连续？
4. 样本空间的维度是多少？低（< 5）、中（5-100）还是高（> 100）？
5. 你需要精确样本还是近似样本？
6. 你需要通过采样操作计算梯度吗？

## 何时使用每种方法

| 方法 | 何时使用 | 复杂度 | 精确？ |
|---|---|---|---|
| 直接采样 | 你有 CDF 或可以使用库函数 | O(1) 每样本 | 是 |
| 逆 CDF | 已知封闭形式 CDF 逆（指数、柯西） | O(1) 每样本 | 是 |
| Box-Muller | 需要不依赖库的正规样本 | O(1) 每样本 | 是 |
| 拒绝采样 | 可以评估目标 PDF，低维（1-3） | O(1/接受率) 每样本 | 是 |
| 重要性采样 | 需要期望而非单个样本 | O(n) 对 n 个样本 | 近似 |
| 分层采样 | 蒙特卡洛估计，希望方差更低 | O(n) 对 n 个样本 | 近似 |
| Metropolis-Hastings | 高维，可以评估非规范化密度 | O(1) 每步 + 预烧期 | 渐近 |
| 吉布斯采样 | 可以从每个条件分布采样 | O(d) 每轮扫描 | 渐近 |
| HMC/NUTS | 高维连续、平滑密度 | O(L * d) 每步 | 渐近 |
| 温度采样 | LLM 文本生成，控制创造力 | O(V) 对词表大小 V | N/A |
| Top-k 采样 | LLM 生成，移除低概率 token | O(V log k) | N/A |
| Top-p（核）采样 | LLM 生成，自适应候选集 | O(V log V) | N/A |
| 重参数化 | 需要通过高斯采样计算梯度（VAE） | O(d) | 是 |
| Gumbel-Softmax | 需要通过分类采样计算梯度 | O(k) 对 k 个类 | 近似 |

## LLM 生成设置

| 使用场景 | 温度 | Top-p | Top-k | 备注 |
|---|---|---|---|---|
| 事实问答 | 0.0（贪婪） | -- | -- | 确定性，无随机性 |
| 代码生成 | 0.2-0.5 | 0.9 | -- | 低创造性，高连贯性 |
| 一般对话 | 0.7 | 0.9 | -- | 平衡 |
| 创意写作 | 0.9-1.2 | 0.95 | -- | 更高多样性 |
| 脑力激荡 | 1.0-1.5 | 0.95 | -- | 最大多样性，可能失去连贯性 |

温度和 top-p 可以组合。先应用温度（缩放 logit），然后应用 top-p 过滤。

## MCMC 方法选择

| 性质 | Metropolis-Hastings | 吉布斯 | HMC/NUTS |
|---|---|---|---|
| 维度 | 任意 | 任意（最好 < 100） | 高（100+） |
| 需要条件分布 | 否 | 是 | 否 |
| 需要梯度 | 否 | 否 | 是 |
| 接受率 | 调至约 23% | 总是 100% | 调至约 65% |
| 相关性 | 高（随机游走） | 中等 | 低 |
| 预烧期 | 长 | 中等 | 短 |
| 最适合 | 探索、简单模型 | 共轭模型、贝叶斯网络 | 连续后验、深度概率模型 |

## 常见错误

- 在高维中使用拒绝采样。接受率随维度指数下降。5 维以上切换到 MCMC。
- MCMC 提议方差设置过高或过低。过 高：大部分提议被拒绝，链卡住。过低：所有提议都被接受，链移动缓慢。随机游走 MH 目标接受率约 23%。
- 忘记预烧期。MCMC 的前 N 个样本受到起始点的偏差。至少丢弃 1000 步（复杂分布可能更多）。
- 使用与目标非常不同的提议进行重要性采样。少数样本获得巨大权重，使估计不可靠。监控有效样本量：ESS = (sum w_i)^2 / sum(w_i^2)。
- 在需要确定性输出的任务中使用温度 > 0（例如分类、结构化提取）。使用贪婪（T=0）或束搜索。
- 不将 top-p 与温度组合。温度alone 不能从长尾中移除垃圾 token。top-p 可以。
- 通过标准采样操作反向传播。对连续（高斯）使用重参数化技巧，对离散（分类）使用 Gumbel-Softmax。

## 快速参考：方差缩减技术

| 技术 | 如何工作 | 方差缩减 |
|---|---|---|
| 分层采样 | 将空间分成层，在每层中采样 | 始终 <= 标准 MC |
| 对偶变量 | 使用 U 和 1-U | 对单调函数有效 |
| 控制变量 | 减去已知均值的变量 | 与相关性成比例 |
| 重要性采样 | 从更好的提议重新加权样本 | 取决于提议质量 |
| 拉丁超立方 | 独立分层每个维度 | 高维优于分层 |
