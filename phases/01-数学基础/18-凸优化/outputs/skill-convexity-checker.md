# 凸性检验器

如何验证优化问题是否凸，以及得到答案后该怎么做。

## 决策检查清单

1. 目标函数是否凸？（检验 Hessian 半正定性或使用复合规则。）
2. 所有不等式约束是否都是 g_i(x) <= 0 的形式，其中每个 g_i 是凸的？
3. 所有等式约束是否都是仿射（线性）的？
4. 如果三个都是是，问题就是凸的。使用有收敛保证的凸求解器。
5. 如果任何一个否，问题是非凸的。使用 SGD/Adam 并接受局部最优。

## 如何检验函数的凸性

| 检验 | 适用范围 | 方法 |
|---|---|---|
| 二阶导数 >= 0 | 标量函数 f(x) | 计算 f''(x)。如果对所有 x 有 f''(x) >= 0，则是凸的。 |
| Hessian 半正定 | 多元函数 f(x) | 计算 H(x)。如果所有特征值在每处 >= 0，则是凸的。 |
| 定义检验 | 任意函数 | 对采样的 x, y, t 检查 f(tx + (1-t)y) <= t*f(x) + (1-t)*f(y)。 |
| 复合规则 | 复合函数 | 参见下方的复合表。 |
| 限制在线上 | 多元 f | f 是凸的当且仅当 g(t) = f(x + tv) 对所有 x, v 在 t 上是凸的。 |

## 复合规则（保持凸性）

| 操作 | 结果 |
|---|---|
| f + g（两个都凸） | 凸 |
| c * f（c > 0，f 凸） | 凸 |
| max(f, g)（两个都凸） | 凸 |
| f(Ax + b)，其中 f 凸 | 凸 |
| g(f(x))，g 凸且非递减，f 凸 | 凸 |
| g(f(x))，g 凸且非递增，f 凹 | 凸 |
| 凸函数之和 | 凸 |
| 凸函数逐点上确界 | 凸 |

## 常见 ML 目标函数：凸还是非凸？

| 目标 | 凸？ | 原因 |
|---|---|---|
| MSE：(1/n) sum(y - Xw)^2 | 是 | 对 w 是二次的，Hessian = (2/n) X^T X 半正定 |
| 逻辑损失：sum(log(1 + exp(-y_i * w^T x_i))) | 是 | log-sum-exp 族的凸函数之和 |
| 铰链损失：sum(max(0, 1 - y_i * w^T x_i)) | 是 | 凸（线性）函数的最大值 |
| L2 正则化：lambda * \|\|w\|\|^2 | 是 | 二次的，Hessian = 2*lambda*I |
| L1 正则化：lambda * \|\|w\|\|_1 | 是 | 绝对值之和（凸但不可微） |
| 岭回归：MSE + L2 | 是 | 两个凸函数之和 |
| LASSO：MSE + L1 | 是 | 两个凸函数之和 |
| 弹性网：MSE + L1 + L2 | 是 | 凸函数之和 |
| SVM（原问题）：铰链 + L2 | 是 | 凸函数之和 |
| 带 softmax 的交叉熵 | 是（在 logit 中） | log-sum-exp 是凸的 |
| 神经网络（任意损失） | 否 | 非线性激活产生非凸复合 |
| k-means 目标 | 否 | 离散赋值步骤 |
| 矩阵分解：\|\|X - UV^T\|\|^2 | 否 | 对 U 和 V 双线性 |
| GAN 损失 | 否 | 极小极大问题，对生成器非凸 |
| 对比损失（InfoNCE） | 否 | 带负样本的指数比值的对数 |

## 基于凸性的求解器选择

| 问题类型 | 求解器 | 收敛保证 |
|---|---|---|
| 凸，光滑，无约束 | 梯度下降 | O(1/k) 到全局最小值 |
| 凸，光滑，无约束 | L-BFGS | 超线性到全局最小值 |
| 凸，光滑，无约束 | 牛顿法 | 在最小值附近二次收敛（如 Hessian 可计算） |
| 凸，光滑，有约束 | 内点法 | 多项式时间 |
| 凸，非光滑（L1） | 近端梯度 / ISTA | O(1/k) 到全局最小值 |
| 凸，非光滑（L1） | ADMM | 灵活，处理约束 |
| 凸，二次 | 共轭梯度 | n 步精确 |
| 非凸，光滑 | SGD / Adam | 收敛到局部最小值 |
| 非凸，光滑 | SGD + 重启 | 平均而言更好的局部最小值 |
| 非凸，光滑 | 过度参数化 + SGD | 平坦最小值，良好泛化 |

## 常见错误

- 因为损失函数是凸的就假设问题是凸的。损失必须在你要优化的参数上是凸的。交叉熵在 logit 中是凸的，但从输入到 logit 的完整神经网络映射是非凸的。
- 在非凸问题上使用牛顿法。Hessian 可能有负特征值，导致牛顿法移动到鞍点或最大值而不是最小值。
- 忘记 L1 正则化使目标函数在零处不可微。标准梯度下降效果不好。使用近端梯度下降或次梯度方法。
- 通过形成 A^T A 使条件数翻倍。如果需要求解最小二乘问题且 A 病态，使用 QR 或 SVD 而不是正规方程。
- 不检验就声称问题非凸。许多 ML 问题（线性模型、SVM、逻辑回归）是凸的，会从更强的求解器中受益。

## 快速检验：我的问题凸吗？

```
1. 写出目标：最小化 f(w) 满足约束
2. 对 f(w) 的每个项：
   - 是 PSD 矩阵的二次？ -> 凸
   - 是范数？ -> 凸
   - 是 log-sum-exp？ -> 凸
   - 涉及 w 非线性（sigmoid(w), w1*w2）？ -> 可能非凸
3. 所有约束是线性的还是凸不等式？
4. 如果所有项都是凸的且约束是凸/线性的 -> 问题是凸的
5. 如果任何项是非凸的 -> 问题是非凸的
```
