# LLM 的差分隐私

> DP-SGD 仍然是标准——噪声 注入梯度更新提供正式（epsilon, delta）保证。计算、内存和效用开销巨大；参数高效 DP 微调（LoRA + DP-SGD）是常见 2025 配置（ACM 2025）。两种证据体紧张：canary-based 成员推断（Duan 等，2024）报告对语言模型有限成功；训练数据提取（Carlini 等，2021；Nasr 等，2025）恢复大量逐字记忆。解决（arXiv:2503.06808，2025 年 3 月）：差距在于测量内容——插入的金丝雀 vs"最可提取"数据。新金丝雀设计无需 shadow 模型即可实现基于损失的 MIA，并在真实数据上对带真实 DP 保证的 LLM 进行首次 nontrivial DP 审计。替代：PMixED（arXiv:2403.15638）——推理时通过专家混合对下一个 token 分布进行私人预测；DP 合成数据生成（Google Research 2024）。新兴攻击：通过 LLM 反馈的差分隐私逆转——置信度分数泄露。

## 问题

LLM 记忆。Carlini 等 2021 表明生产语言模型按需再现逐字训练文本。DP 是正式防御：训练使输出provably 对任何单个训练样本不敏感。2024-2025 年证据显示 DP-SGD 是必要的但部署的 ε 值可能不匹配威胁模型。

## 概念

### （ε, δ）-差分隐私

随机算法 M 是（ε, δ）-DP，如果对于任何相差一个示例的两个数据集和任何事件 S：
P(M(D) in S) <= e^ε * P(M(D') in S) + δ。

解释：输出分布足够接近（由 ε 参数化），任何单个个体的贡献不能被可靠推断，除概率 δ 外。

### DP-SGD

Abadi 等 2016。标准方法：
1. 采样小批量。
2. 计算每个示例梯度。
3. 将每个示例梯度裁剪到阈值 C。
4. 求和裁剪梯度并添加 std σ * C 的高斯噪声。
5. 使用噪声和更新参数。

隐私成本由计算器（矩计算器、Rényi DP 计算器）跟踪。LLM 文献中发布的 ε 值因威胁模型、数据敏感性和效用目标差异很大；没有普遍"安全"默认 ε。发布示例在某些 LLM 训练设置中大约 ε ≈ 1–10，但这些是说明性的——不是推荐默认。较低 ε 通常需要更多噪声且会增加效用损失。

### LoRA + DP-SGD

完整 DP-SGD 的前沿模型禁止。LoRA（Hu 等 2022）将梯度更新限制在小适配器，减少每示例梯度存储。LoRA + DP-SGD 是 2025 年常见配置。DP 保证适用于适配器；基础模型保持固定。

### 2024-2025 年紧张

两条证据线：

- **Canary MIA（Duan 等 2024）。** 将唯一金丝雀插入训练数据，测量成员推断攻击者是否能识别它们。报告对语言模型有限成功。提示 MIA 很难。
- **训练数据提取（Carlini 2021、Nasr 等 2025）。** 用前缀提示模型；测量是否恢复训练的逐字文本。报告大量记忆。提示 MIA 很容易。

2025 年 3 月解决（arXiv:2503.06808）：两者测量不同东西。MIA 问"示例 e 在 D 中吗？"在插入的金丝雀上。提取问"我能恢复 D 的什么？" "最可提取"示例才是重要的；金丝雀低估此因为它们不是为了可提取性而优化的。

新金丝雀设计。无 shadow 模型的基于损失的 MIA。在真实数据上对 LLM 进行首次 nontrivial DP 审计，带真实 DP 保证。

### DP 训练的替代方案

- **PMixED（arXiv:2403.15638）。** 推理时私人预测。每个专家看到一部分训练数据；聚合添加噪声以满足 DP。完全避免 DP 训练。
- **DP 合成数据生成（Google Research 2024）。** 用 DP-SGD 的 LoRA 微调，采样合成数据，在合成数据上训练下游分类器。

两者在完全 DP 训练的效用成本下回避了不同威胁模型。

### 通过 LLM 反馈的差分隐私逆转

新兴 2025 年攻击。使用 DP 训练模型的置信度分数作为预言机重新识别个体。即使输出不泄露，置信度分布也可以。

防御：不暴露置信度，或在暴露前截断/量化。这是（ε, δ）-DP 训练之外的附加要求。

## 关键术语

| 术语 | 常见说法 | 实际含义 |
|------|---------|---------|
| DP | （ε, δ）-差分隐私 | 正式隐私：相邻数据集变化的输出分布接近 |
| DP-SGD | 噪声注入 SGD | 梯度裁剪 + 高斯噪声添加；标准 DP 训练 |
| LoRA + DP-SGD | 高效私人微调 | 在低秩适配器上的 DP-SGD；2025 年标准配置 |
| MIA | 成员推断 | 确定示例是否在训练数据中的攻击 |
| 金丝雀 | 插入水印示例 | 用于测量 DP 泄露的唯一训练示例 |
| PMixED | 私人推理混合 | 通过下一个 token 分布上的专家混合进行推理时 DP |
| DP 逆转 | 置信度泄露攻击 | 使用模型置信度作为预言机进行重新识别的攻击 |
